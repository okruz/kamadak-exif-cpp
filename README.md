# kamadak-exif-cpp

## How to build
First, you need to define the output path for the output path for library header generated by cbindgen:
```
export CBINDGEN_OUTPUT=`pwd`/kamadak_exif_cpp.h
```
Afterwards, you can compile with `cargo build`.
